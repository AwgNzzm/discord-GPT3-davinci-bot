# discord-GPT3-davinci-bot

Just 
- `git clone https://github.com/ediziks/discord-GPT3-davinci-bot.git`
- `cd discord-GPT3-davinci-bot`
- `pip3 install -r requirements.txt`
- `nano .env`

and create two variables as;
```
GPT3_TOKEN=
DISCORD_TOKEN=
```

that's all. 
now just need to run
- `python3 bot.py`
